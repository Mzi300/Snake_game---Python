# Python Snake Game

## Description
This is a classic **Snake game** implemented in Python using the **Pygame** library.  
The project demonstrates basic game development concepts such as game loops, collision detection, user input handling, and score tracking.

## Status
⚠️ Work in Progress  
The project is currently under development and may not be fully functional yet.

## Features (Planned)
- Playable snake controlled with arrow keys
- Increasing difficulty as the snake eats more food
- Score tracking
- Game over detection
- Simple and interactive Pygame interface

## Requirements
- Python 3.x
- `pygame`
